1 HTML-da <div> va <span> elementlari orasidagi farq nima va qachon ularni qo'llash kerak

  <div> - блочный тег, занимает всю ширину экрана. Один из важных тегов при помощи которого строится вся структура сайтов. Является парным тегом

  <span> - строчный тег, строчные теги в отличии от блочных тегов занимают только ту часть которая им нужна тоесть сколько контетна внутри столько они и занимают, еще одно отличие от блочнчых тего к строчным тега не применяются некоторые css стили. Является парным тегом 

2 CSS box model’i nima?

  Свойство влияет на поведение размеров блока, к примеру если мы задаем падинг справа блоку то его ширина увеличится, это повляет на верстку чтобы такого не было и размер оставался как есть но при этом у него был паддинг то можно задать значение border-box 

3 'this' kalit so'zi nima va u qanday ishlaydi? Misollar bilan tushuntiring.

  this - это контекст, он обращается к названию обьекту в котором он лежит. Например:

    const obj = {
      firstName: 'user',
      lastName: 'userov'
      age: 25,
      getFullName() {
        return `Полное имя пользователя ${this.firstName} ${this.lastName}`
      }
    }
  В нашем случае обращение к имени и фамиилии через контектс this будет равноправен записи 
  obj.firstName obj.lastName
